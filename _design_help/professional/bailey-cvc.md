Mitch provides physical verification support for open-source GF180MCU projects, focused on diagnosing and resolving LVS/ERC failures and helping teams tapeout with confidence.

He's the developer of **CVC-RV**, an open-source verification system for full-chip parameter-based reliability and electrical rule checks (ERC+).

### Scope
- LVS/ERC error investigation
- Verification flow and rule configuration
- PDK/integration-related issue diagnosis

- One 60-minute exploration session  
  (investigation and limited experiments; resolution not guaranteed)

### Ongoing Support
- Hourly support for continued debugging or configuration work
